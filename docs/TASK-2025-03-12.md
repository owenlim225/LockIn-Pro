# Tasks for 2025-03-12

## 1. Bug: Mark done stops working after first task with photo

- **Scope:** Trace the completion flow (proof modal, submit, state update) and fix so multiple tasks can be completed with proof in one session.

## 2. Good vs bad habit distinction on Quest and Stats

- **Quest path tab:** Show a clear distinction between good habits and bad habits (e.g. separate sections, labels, or visual treatment so "Avoid" / bad habits are obvious).
- **Statistics tab:** Same distinction—good vs bad habits should be visually or structurally separated (e.g. sections, labels, or filters).

## 3. Manage habits: actions above the list

- **Change:** On the Manage habits view, move the **Export**, **Reset**, and **Delete habits** buttons **above** the habit cards list (so they appear before the cards, not below or in a different order).
- **Goal:** Actions are visible and accessible before scrolling through habits.

---

## 4. Feature: Completion celebration popup (save/share screenshot)

- **Behavior:** After the user completes **all** tasks for the day, show a popup that allows them to **save/download** or **share** a screenshot.
- **Screenshot content:** Number of tasks completed, the date, and the current streak.
- **Goal:** Let users celebrate and share their daily completion.

---

## 5. Bug: Overdue styling only when reminder or deadline exists

- **Issue:** Goals that have **no** reminder or deadline are currently affected by the overdue color change.
- **Fix:** Apply the overdue color/style **only** to goals that have a reminder or deadline. Goals without either should not show overdue styling.

---

## 6. Bug: Light mode and dark mode color switching

- **Issue:** Light mode and dark mode are not working properly; page colors do not change correctly when switching themes.
- **Fix:** Ensure theme switching updates the page colors correctly so that light mode and dark mode each display the appropriate palette across the app.

## 7. Future (2nd priority): Shared room / group goals

- **Priority:** Second priority for future development.
- **Concept:** A **room** that other people can join, with shared goals and activity.

**Tabs:**

- **Feed tab:** Status report / status log of everything happening in the room (e.g. someone new joining, someone failing a task, someone completing a task), each entry with a **timestamp**.
- **Goal tab:** All goals that **all members** of the room will complete (shared room goals).
- **Rank tab:** Leaderboard for members—who completes tasks the most (ranking by completions).
- **Members tab:**
  - Invite code and/or **shareable link** to invite friends.
  - List of **all members** of the room with:
    - Name
    - Number of contributions
    - Task completed tracker (e.g. **0/5**, **1/5**, etc.)
  - **Leave room** button at the bottom (below the member list).

---
